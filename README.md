# UTS Enterprise Application Integration (EAI) - Service to Service

Proyek ini merupakan implementasi arsitektur **Service-to-Service** yang dikembangkan untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Enterprise Application Integration (EAI). Sistem ini membagi fungsionalitas menjadi beberapa layanan terpisah (User, Product, dan Order) yang saling berkomunikasi.
